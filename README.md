## 主要依赖

- Jetpack Compose：安卓UI框架
- Hilt：依赖注入
- Material3：组件库
- Android Navigation：导航
- Room：本地数据库
- Coil：图片加载
- mmkv：腾讯开发的本地快速存储工具
- okhttp：网络请求
- moshi：序列化和反序列化
- Biometric：安卓官方生物识别库(带UI的指纹识别和密码验证)
